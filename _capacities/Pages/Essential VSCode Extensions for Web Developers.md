# Essential VSCode Extensions for Web Developers

## UI

- [GitHub Copilot](https://marketplace.visualstudio.com/items?itemName=GitHub.copilot) -> real-time code suggestions using OpenAI, including writing code for you based on a prompt.

- [GitLens](https://marketplace.visualstudio.com/items?itemName=eamodio.gitlens) -> enhance Git version control capabilities, like navigating Git repos.

- [VS Code Icons](https://marketplace.visualstudio.com/items?itemName=vscode-icons-team.vscode-icons) -> customize the look of specific files and file types.

- [Relative Path](https://marketplace.visualstudio.com/items?itemName=jakob101.RelativePath) -> provides the relative path for the files in your workspace.

## Basic Formatting

- [Prettier](https://marketplace.visualstudio.com/items?itemName=esbenp.prettier-vscode) -> makes your code pretty by automatically formatting it when you save the file.

- [Code Spell Checker](https://marketplace.visualstudio.com/items?itemName=streetsidesoftware.code-spell-checker) -> helps you find and correct common spelling errors.

- [Indent Rainbow](https://marketplace.visualstudio.com/items?itemName=oderwat.indent-rainbow) -> colors indents so you can see them better.

- [Better Comments](https://marketplace.visualstudio.com/items?itemName=aaron-bond.better-comments) -> improve your code comments for readability.

## HTML

- [Auto Rename Tag](https://marketplace.visualstudio.com/items?itemName=formulahendry.auto-rename-tag) ~~-> automatically rename opening/closing HTML tags when one is edited.~~

- [Auto Close Tag](https://marketplace.visualstudio.com/items?itemName=formulahendry.auto-close-tag) ~~-> automatically add a closing HTML tag when you type the opening one.~~

## CSS

- [Color Highlight](https://marketplace.visualstudio.com/items?itemName=naumovs.color-highlight) -> visualizes hex code colors in your CSS.

- [CSS Peek](https://marketplace.visualstudio.com/items?itemName=pranaygp.vscode-css-peek#:~:text=Peek%3A%20load%20the%20css%20file,the%20symbol%20(%20Ctrl%2Bhover%20)) -> quickly view CSS style definitions for classes and IDs that are defined within the workspace.

- [Intellisense for CSS](https://marketplace.visualstudio.com/items?itemName=Zignd.html-css-class-completion) -> provides CSS class completion from those defined in the workspace.

## JavaScript

- [JavaScript Booster](https://marketplace.visualstudio.com/items?itemName=sburg.vscode-javascript-booster) -> helps you perform common refactoring tasks in JavaScript.

- [ESLint](https://marketplace.visualstudio.com/items?itemName=dbaeumer.vscode-eslint) -> integrates ESLint, a tool that finds and fixes problems in Javascript code, with VSCode.

- [JavaScript (ES6) Code Snippets](https://marketplace.visualstudio.com/items?itemName=xabikos.JavaScriptSnippets) -> provides useful code snippets for JavaScript using ES6 syntax.

## Markdown

- [Markdown All in One](https://marketplace.visualstudio.com/items?itemName=yzhang.markdown-all-in-one) -> provides all your markdown needs, including keyboard shortcuts, auto preview files, autocompletion, and table of contents.
